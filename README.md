# ACF Image Point

A WordPress plugin that extends Advanced Custom Fields (ACF) with an image point field type.

## Description

This plugin allows you to add interactive point markers on images within ACF field groups.

## Requirements

- WordPress 5.0 or higher
- Advanced Custom Fields (ACF) 5.0 or higher

## Installation

1. Upload the plugin files to `/wp-content/plugins/acf-image-point/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a new ACF field and select "Image Point" as the field type

## Usage

Add an Image Point field to your field group and configure the settings. Users can then click on an image to place point markers.
